# .vimrc
This is my .vimrc

## Supporting OS/dist
- macOS Catalina
- Ubuntu 18.04
- Ubuntu 20.04

## Installation
```bash
git clone git://github.com/nemumu/vimrc.git

# Vim
cp vimrc/.vimrc ~/

# Neovim
cp vimrc/init.vim ~/.config/nvim/init.vim
```

### Ubuntu
#### vim
```bash
sudo apt install mypy flake8 vim-python-jedi
```

#### Neovim
```bash
sudo apt install fzf ripgrep silversearcher-ag
pip3 install pyright
```
